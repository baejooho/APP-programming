# 퀴즈 최상급 표현 검증

`data/questions.json`을 읽고, 문제(`question`) 또는 보기(`options`) 텍스트에
아래 최상급 표현이 포함된 항목을 모두 찾아 목록으로 출력해 줘.

**검색 대상 표현:** 가장, 최초, 최대, 최소, 최고, 최장, 최단, 최강, 최초, 유일, 처음, 제일

**출력 형식:**
각 문제마다 아래 정보를 표시해 줘:
- 문제 ID
- 카테고리 / 난이도
- 문제 텍스트 (해당 표현 강조)
- 감지된 표현
- 가이드라인 판정: 기준 명시됨 / 기준 미명시 / 논란 가능성

판정 기준:
- 문제 텍스트나 hint에 측정 기준(예: 해발 고도, 면적, 길이 등)이 명시되어 있으면 → **기준 명시됨**
- 기준 없이 최상급 표현만 있으면 → **기준 미명시 (수정 권장)**
- 학계·출처 간 논란이 알려진 사실이면 → **논란 가능성 (explanation 보완 권장)**

목록 출력 후, 수정이 필요한 항목이 있으면 구체적인 수정 제안도 함께 제시해 줘.
