# 퀴즈 범위 검토

`data/questions.json`을 읽고, **$1번 인덱스부터 $2번 인덱스까지**의 문제(0-based)를 검토해 줘.
인수가 없거나 비어 있으면 파일 전체 문제를 검토해 줘.

---

## 1. 문제 목록

각 문제마다 아래 정보를 표시해 줘:

| 항목 | 내용 |
|---|---|
| 번호 | 파일 내 순서 (1-based) |
| ID | 문제 ID |
| 카테고리 | category 필드 |
| 난이도 | difficulty 필드 |
| 문제 | question 필드 |
| 정답 보기 | options[answer] 값 (answer 인덱스 번호도 함께) |

---

## 2. 난이도 분포

해당 범위 내 문제의 난이도(초급 / 중급 / 고급)별 개수와 비율을 표로 출력해 줘.

예시:
| 난이도 | 개수 | 비율 |
|---|---|---|
| 초급 | 3 | 50% |
| 중급 | 2 | 33% |
| 고급 | 1 | 17% |

난이도 균형에 문제가 있으면(한 난이도가 60% 초과) 경고를 표시해 줘.

---

## 3. 정답 분포

해당 범위 내 문제의 정답 인덱스(0~3) 분포를 표로 출력해 줘.

예시:
| 정답 위치 | 보기 번호 | 개수 | 비율 |
|---|---|---|---|
| 0 | 1번 | 2 | 33% |
| 1 | 2번 | 1 | 17% |
| 2 | 3번 | 2 | 33% |
| 3 | 4번 | 1 | 17% |

정답이 특정 위치에 편중되어 있으면(한 위치가 40% 초과) 경고를 표시해 줘.

---

## 4. 종합 의견

- 검토한 문제 수, 카테고리 구성 요약
- 난이도 균형 평가
- 정답 편중 평가
- 개선이 필요한 문제가 있으면 구체적으로 명시해 줘
