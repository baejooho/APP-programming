# PRD: 상식 퀴즈 게임

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 상식 퀴즈 게임 |
| 작성일 | 2026-03-31 |
| 목적 | 사용자가 다양한 분야의 상식을 재미있게 학습하고 실력을 확인할 수 있는 퀴즈 게임 제공 |
| 타겟 사용자 | 성인 |

---

## 2. 목표

- 4개 카테고리(한국사, 과학, 지리, 일반상식)에 걸쳐 총 40문제 제공
- 즉각적인 피드백으로 학습 효과 극대화
- 점수 및 순위 기록을 통해 재참여 동기 부여

---

## 3. 사용자 흐름

```
시작 화면
  └─> 닉네임 입력
        └─> 카테고리 선택 (한국사 / 과학 / 지리 / 일반상식 / 전체)
              └─> 난이도 선택 (초급 / 중급 / 고급)
                    └─> 퀴즈 진행 (10문제 또는 40문제)
                          └─> 문제 화면 → 답 선택 → 즉시 피드백
                                └─> 최종 결과 화면 (점수 + 순위)
                                      └─> 재도전 / 종료
```

---

## 4. 기능 요구사항

### 4.1 퀴즈 문제

| 항목 | 내용 |
|------|------|
| 카테고리 | 한국사, 과학, 지리, 일반상식 |
| 문제 수 | 카테고리별 10문제 × 4 = 총 40문제 |
| 문제 형식 | 4지선다 객관식 |
| 난이도 | 초급 / 중급 / 고급 (게임 시작 전 선택, 선택한 난이도 문제만 출제) |
| 문제 순서 | 랜덤 셔플 |
| 보기 순서 | 랜덤 셔플 |

### 4.2 게임 진행

- 문제 1개씩 순차 표시
- 4개의 보기 중 1개 선택
- 제한 시간: 문제당 20초
- 타임오버 시 오답 처리
- 힌트 기능: 문제당 1회 사용 가능, 사용 시 해당 문제 점수 -50%

### 4.3 즉시 피드백

- 정답 선택 시: 초록색 하이라이트 + "정답입니다!" 메시지
- 오답 선택 시: 빨간색 하이라이트 + 정답 표시 + 간단한 해설
- 다음 문제로 이동 버튼 제공

### 4.4 점수 시스템

| 조건 | 점수 |
|------|------|
| 정답 - 초급 | +5점 |
| 정답 - 중급 | +10점 |
| 정답 - 고급 | +15점 |
| 빠른 정답 (7초 이내) | +5점 추가 보너스 |
| 힌트 사용 후 정답 | 해당 문제 점수 × 50% |
| 오답 | 0점 |

### 4.5 순위 기록

- 닉네임 + 점수 + 날짜 저장
- 상위 10명 리더보드 표시
- 카테고리별 / 전체 순위 구분
- 로컬 저장 (파일 또는 로컬 DB)

---

## 5. 화면 구성

### 5.1 시작 화면
- 게임 타이틀
- 닉네임 입력 필드
- 카테고리 선택 버튼 (5개: 개별 4 + 전체)
- 리더보드 보기 버튼

### 5.2 난이도 선택 화면
- 선택한 카테고리 표시
- 난이도 선택 버튼 (초급 / 중급 / 고급)
- 난이도별 점수 안내

### 5.3 퀴즈 화면
- 진행 상황 표시 (예: 3/10)
- 카테고리 / 난이도 / 현재 점수 표시
- 타이머 바 (20초)
- 문제 텍스트
- 4개 보기 버튼
- 힌트 버튼

### 5.4 피드백 화면
- 정답/오답 여부
- 정답 강조 표시
- 해설 텍스트
- 다음 문제 버튼

### 5.5 결과 화면
- 최종 점수 / 만점 대비 비율
- 카테고리별 정답률
- 순위 및 리더보드 위치
- 재도전 / 카테고리 변경 / 종료 버튼
